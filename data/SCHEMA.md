# SCHEMA — OnomaDict ベクトル構造定義

`data/onomatopoeia_dictionary.csv`（764語）の列定義。ベクトルは V = [A / B / C / D]、コア16次元（x₁–x₁₆）＋IPA・派生・メタ列。

## 認識論的前提（重要）
本辞書の「物理的」な軸は身分が二種に分かれ、混同しない。
- **素朴物理学（naïve physics）**：人が知覚・実演する運動の質。観察者依存の物理像。→ Category A。
- **本物物理学（real physics）**：実在の物理現象・測定。→ Category B（音響測定）、Category C の Reynolds 等（口腔内空気の実流体力学。ただし出力は音象徴イメージの強化に奉仕）。

距離計算や学習で A と B/C を同一の「物理軸」として等価に扱わないこと。

## 正規化原則
A軸は、脱力して重力に任せた受動的運動（自由落下・振り子）を基準（0方向）とし、そこからの逸脱で評価する（KSD正規化）。特に Weight・Flow。

## Category A：ラバン・エフォート（身体表現系／素朴物理学）

|---|---|---|---|
| `x1` | Weight | 0:軽い ↔ 9:重い | 動力学質。込めた力・重み。濁音・/a,o/＝重い側。x7（低周波）と混同しない |
| `x2` | Time | 0:持続的 ↔ 9:突発的 | **時間への態度**（即決か時間に委ねるか）。音の長さ・包絡ではない。促音っ／長音ーは手がかりだが定義ではない。音響時間は Category B（x8）に分離 |

## 現状と既知の限界
- **x₁–x₁₂ は確定値**。x₂ は「時間への態度」に定義を純化済み（意味・極性は不変）。
- **D軸（x₁₃–x₁₆）は現在、形態型からの一括ブートストラップ値**であり、離散に固着している（x16∈{0,1,8}、x13 の多くが既定4、x14 の多くが5）。連続軸として機能させるため、合成音エンベロープからの実測較正を予定：IOI変動係数→x16、エネルギー時間重心→x13、振幅トレンド→x14、オンセット密度→x15。単発語は形態priorを維持。
- x₁₆ と x₉ は独立（764語で r=−0.16）。
- x₉ は乱流側に偏在（x9_norm 中央値≈6.5）。層流側サンプルが手薄。
