## 2. 本質的論点：離散固着の根本解＝実音エンベロープ測定
形態型は有限個なので「形態型→離散値」のブートストラップは必ず値が固まる。これは初期状態として正しいが、連続軸化には実測較正が要る。Note B の提案を全面採用。各D軸の音響特徴を以下に確定する。

- **x₁₆ Regularity** ← オンセット間隔（IOI）の変動係数 CV = std(IOI)/mean(IOI)。単発（オンセット1個）は律動未定義→0で固定（縮退値）。CVを0–9へ線形マップ（CV≈0→0、CV≈0.6以上→9程度を上限に較正）。
- **x₁₃ Accent** ← 強度エンベロープの時間重心（energy centroid）を相内normalized時間[0,1]で取り、×9。前寄り=impulse(低)、後寄り=impact(高)。437語の既定4はこれで上書き。
- **x₁₄ Contour** ← 各オンセットのピーク振幅列（または IOI 列）の傾き。増勢/短縮=加速(低)、減勢/伸長=減速(高)、平坦=5。
- **x₁₅ Meter** ← オンセット密度（単位時間あたりオンセット数）。単発0、連続ドローンが上限9。回数より密度で連続化。

運用：morph_type 由来の値は **prior（事前値）** として保持し、合成音が取れる語は実測値で **上書き**。実測不能・単発語は prior 維持。これで3値→連続分布になる。

## 3. 三つの質問への決定

### Q1. x₁₆/x₉ の閾値（x9_norm=4.5、x₁₆=2/7）
**象限の閾値は表示・検証用の便宜であり、軸定義には含めない。** 独立性の根拠 r は値から計算され閾値非依存なので、結論は境界をどこに引いても不変。よって「私のジッター定義に合わせ直す」必要はない（私の定義は IOI-CV であり、そもそも閾値を持たない連続量）。
推奨は2点だけ：
- x₁₆ 側：現状の 2/7 は {0,1,8} の隙間に置いた人工物。連続化後はスケール中点 **4.5** に移す。
- x₉ 側：物理的な層流/乱流遷移（Re≈2300）は、そちらの正規化（x9_norm=(log10Re−2)/2.301×9）では **x9_norm≈5.3** に対応。表示境界を 4.5→**5.3** に寄せると物理的に正当で、乱流偏在とも整合（層流側が手薄なことが正しく見える）。
いずれも検証図の見栄えの話で、移行をブロックしない。

### Q2. x₁₅ Meter に3回以上反復（ちんちろりん等）の別水準を設けるか
**離散の別ビンは作らない。** 代わりに x₁₅ を「反復回数」から「オンセット密度」由来に切り替え、3回以上は密度が上がる分だけ自然に高い値になるよう連続化する（2×畳語より3+反復・トリル系が上、連続ドローンが上限）。ブートストラップ段階だけは暫定で 2×→5 / 3+→6–7 と段差を付けてよいが、最終は §2 の密度測定で吸収。
重要：**3+形態（ちんちろりん, けちょんけちょん, ぴーちく等）は内部に複数オンセットと固有のアクセント・推移を持つ「フレーズ」であり、D軸が最も効く語群。** これらを実音測定の優先対象に回してほしい（x₁₃/x₁₄/x₁₆ がここで初めて意味を持つ）。

### Q3. 3点キーフレーム補間規則（A軸確定待ち）
A軸は §0 で確定。よって補間規則を確定する。**そして §2 の包絡測定がそのままキーフレーム幾何を与える**（時間重心→τ、振幅トレンド→非対称）。一回の音響解析で較正とキーフレームの両方が出る。

相を normalized 時間 [0,1] とし、各質 q∈{x₁,x₂,x₃,x₄} を [t0, t1, t2] の3点へ展開：

1. **ピーク時刻 τ** = 0.1 + 0.8·(x₁₃/9)。実音があれば τ = エンベロープの時間重心。t1 を τ に置く。
2. **アクセント点 t1 の値** = 注釈済み x_q（静的値＝ピーク）。
3. **立ち上がり/減衰の非対称** は x₁₄：
   - 加速 (x₁₄<5)：t0 低 → t1 ピーク（溜めて上げる）、t2 はピーク維持寄り。
   - 減勢 (x₁₄>5)：t0 はピーク近く → t2 で大きく低下（放って減衰）。
   - 一定 (5)：対称・浅い山。
4. **質ごとの追従**：
   - x₁ Weight・x₂ Time はアクセントで最大（ピーク＝注釈値）。
   - x₄ Flow は既定で Weight に追従。ただし「制御された打撃」型は τ で一旦 bound 側へ締まり、直後に free へ解放（語義フラグで選択）。← ここだけ要・合同決定。
   - x₃ Space は τ で最も direct、両端で緩む浅い山。
5. **反復**：x₁₅>0 なら3点単位を反復。周期 P は x₁₅（密度）から、P と各単位ピークのばらつきは x₁₆（ジッター）で付与。x₁₅→9 で単位が融合し連続ドローン（3点が平坦化）。

唯一の要合同決定は 4 の「Flow がアクセントで締まるか追従か」の語義フラグ運用のみ。他は機械化可能。
